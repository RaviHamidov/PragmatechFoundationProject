# AlgorithmTasks with JavaScript 
## Content
```
    1. user tərəfindən daxil edilən sözün boyuk herflerini kicikle, kicik herflerini böyüklə yazmaq.
    2. user tərəfindən daxil rəqəmin tam və kəsr hissəsinə ayırıb, toplamaq.
    3. user tərəfindən daxil 2 ədədin 30-70 arasında olmasını yoxlamaq. Rəqəmlərin hər ikisinin, yalnız birinin və ya heç birinin aralıqda olub-olmaması şərtlərini yoxlamaq və uyğun mesajı çıxarmaq.
    4. Verilmiş arreyin uzunluğunu elementlərini çıxarmaq. numbers = [45, 4, 9, 16, 25];
    5. 2 ayrı arreyi birləşdirmək. 
    6. Rənglərə aid arrey yaradib arrey-ə kənardan əlavə söz əlavə etmək.
    7. Yeni bir obyekt yaradib onun elementlərini console log da çıxarmaq. (Məsələn, Taski şəxsin adı Aydan, Yaşı  qadının yaşı soruşulmaz, vəzifəsi Tələbə koordinatorudur)
    8. istənilən 2 parametri (rəqəmi) toplayan function yaratmaq.
    9. user-ın daxil etdiyi sözün sonuncu hərfini qaytarmaq.
```
